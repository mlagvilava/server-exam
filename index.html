Windows Server 2022-ის ინსტალაცია

01

დისკის დაყოფა C: და D: — Windows Server 2022 დაყენება

VMware / VirtualBox → ISO ფაილი → ინსტალაცია
📸 სქრინი
1
ვირტუალური მანქანის გამართვა

სანამ დააყენებ, ვირტუალური მანქანა მომზადებული უნდა გქონდეს:

→გახსენი VMware Workstation ან VirtualBox
→შექმენი ახალი ვირტუალური მანქანა → OS: Windows Server 2022
→RAM: მინიმუმ 4 GB; CPU: მინიმუმ 2 core; Disk: მინიმუმ 100 GB
→CD/DVD Drive-ში: მიუთითე Windows Server 2022 ISO ფაილი
→გაუშვი მანქანა — რამდენიმე წამში გამოჩნდება ინსტალაციის ეკრანი
2
ინსტალაციის პირველი ეკრანი
💡ეკრანზე ნახავ Windows Setup ფანჯარას. ყველაფერი ინგლისურად იქნება — ეს ნორმალურია.
→Language to install: English (United States) — ნუ შეცვლი
→Time and currency format: English (United States)
→Keyboard: US → Next
→დიდ ლურჯ ღილაკს ნახავ: Install Now → დააჭირე
3
OS ვერსიის არჩევა
→სიაში ნახავ რამდენიმე ვარიანტს → აირჩიე: Windows Server 2022 Standard Evaluation (Desktop Experience)
→Next
→License terms: ✅ მონიშნე „I accept the Microsoft Software License Terms" → Next
ℹ️Desktop Experience — ნიშნავს რომ გექნება ჩვეულებრივი სამუშაო მაგიდა (Desktop). ამ ვარიანტი აუცილებლად აირჩიე — სხვა ვარიანტებს არ აქვს ვიზუალური ინტერფეისი.
4
ინსტალაციის ტიპის არჩევა
→გამოჩნდება ორი ვარიანტი → აირჩიე: Custom: Install Microsoft Server Operating System only (advanced)
⚠️„Upgrade" ნუ აირჩიე — ის წინა ვერსიიდან განახლებისთვისაა. ჩვენ ახლიდან ვაყენებთ.
5
დისკის დაყოფა — C: (OS) და D: (Data)
📌ახლა ხარ „Where do you want to install the operating system?" ეკრანზე. ნახავ ერთ ჩანაწერს: Drive 0 Unallocated Space (მთელი ცარიელი დისკი).

ნაბიჯი 5.1 — C: დისკის შექმნა (OS-ისთვის):

→მონიშნე Drive 0 Unallocated Space
→ქვემოთ ნახავ ღილაკებს → დააჭირე New
→Size ველი გამოჩნდება MB-ში. C: = 60GB → ჩაწერე 61440 (60 × 1024 = 61440) → Apply
→Pop-up: „Windows might create additional partitions for system files" → OK
⚠️Windows ავტომატურად შექმნის 500 MB System Reserved partition-ს. ეს ნორმალურია და ხელი ნუ ახლე — ამ partition-ს Windows-ის გასაშვებად სჭირდება.

ნაბიჯი 5.2 — D: დისკის შექმნა (Data-სთვის):

→სიაში კვლავ ნახავ Unallocated Space (დარჩენილი ადგილი) → მონიშნე
→New → Size: ნუ შეცვლი — ჩაყარე ყველაფერი → Apply
Partitionზომასახელიდანიშნულება
Partition 1~500 MBSystem ReservedWindows Boot (ნუ შეეხები)
Partition 2 → C:~60 GBPrimaryWindows Server OS
Partition 3 → D:დარჩენილიPrimaryFiles, Data, Shares

ნაბიჯი 5.3 — Windows-ის C: დისკზე დაყენება:

→სიაში Partition 2 (C:, ~60GB) მონიშნე → Next
→ინსტალაცია დაიწყება ავტომატურად — 10–25 წუთი გასტანს, სერვერი ორ-სამჯერ გადაიტვირთება
→გადატვირთვის შემდეგ: ეკრანზე გამოჩნდება „Customize settings" → ჩაწერე Administrator-ის პაროლი (მაგ: Admin@2024!) → Finish
✅შედეგი: Windows Server 2022 ჩაიტვირთება. Server Manager გაიხსნება ავტომატურად.
📸 სქრინი: Server Manager გახსნილი ეკრანი — გჩანდეს სერვერის სახელი და Dashboard
